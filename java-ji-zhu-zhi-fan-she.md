## 反射

Java中的反射技术就相当于一面镜子，可以用这面镜子把指定的对象内部所包含的东西统统都“照映”出来。反射可以在运行状态下把一个 Class 的内部成员方法、成员变量和构造函数全部获取到，并且能够进行实例化、直接调用内部的成员方法。

- 注意的点: 

```
  method.setAccessible(true); // 如果是 private 的方法，需要获取其调用权限
```
